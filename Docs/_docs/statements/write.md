# Write

{: .example }
> Write a black pixel:
>
> ```tmlx
> write black
> ```

## Write - Definition and Usage

The `write` function writes a color.

## Write - Syntax

```ebnf
write color
```

## Write - Fields

Field | Description
-- | --
color | Required. `0`/`white`, `1`/`black`, `blue`, `cyan`, `green`, `red`, `magenta` or `yellow`.

## Write - More Examples

{: .example }
> Emphasize binary:
>
> ```tmlx
> write 1
> ```
